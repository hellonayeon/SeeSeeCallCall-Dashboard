## 소개
드로잉투게더 모바일 앱의 사용 현황을 실시간으로 보여주는 대시보드입니다.  
  
웹 서비스를 이용하여 모니터링한 데이터를 관리자에게 실시간으로 제공합니다. 차트를 통해 모니터링으로 수집한 데이터들을 시각적으로 보여줍니다.
